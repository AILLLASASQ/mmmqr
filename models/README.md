# مجلد النماذج (.glb)

ضع ملفاتك هنا بنفس الأسماء المشار إليها في index.html:

- character.glb                ← الشخصية (يُفضّل أن تحتوي أنميشن مشي)
- buildings/b1.glb ... b8.glb  ← المباني
- products/chair.glb, vase.glb, lamp.glb, table.glb, statue.glb  ← المنتجات

لتغيير الأسماء/الأعداد: عدّل المصفوفات CHARACTER / BUILDINGS / PRODUCTS أعلى السكربت في index.html.

مصادر النماذج: Blender، Sketchfab، أو أدوات توليد 3D (Meshy / Tripo / Luma) ثم صدّر .glb.
